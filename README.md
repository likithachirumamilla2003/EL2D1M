Excel Analysis using Pivot Tables 
Dataset Used:
Superstore Sales Dataset
Objective:
To analyze sales performance using Excel Pivot Tables and derive meaningful business insights from the data.
Work Performed:
Converted the dataset into an Excel Table to enable dynamic analysis.
Created a Pivot Table to analyze total sales by category, identifying the highest and lowest performing categories.
Built a Region vs Segment Pivot Table to compare sales performance across Consumer, Corporate, and Home Office segments.
Applied conditional formatting to visually highlight high- and low-performing regions.
Added slicers (Category, Region, Segment) to enable interactive filtering and analysis.
Created a bar/column chart to visually represent sales performance across regions and segments.
Summarized findings into five key insights based on pivot analysis.
